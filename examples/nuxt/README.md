# Nuxt Example

This app demonstrates `@xcvzmoon/vue-cropperjs` in Nuxt with the recommended SSR-safe pattern.

It shows:

- a client-only wrapper component for `CropperImage`
- an app composable wrapper for Nuxt auto-imports
- `v-model:data` state syncing
- exporting the selection to a preview image

## Run

```bash
pnpm install
pnpm dev
```
